# ShoppyGlobe E-Commerce Application

A React based e-commerce application built using Vite.

## Technologies Used

- React
- Vite
- React Router
- Redux Toolkit
- React Redux
- CSS


## Features

- Product listing
- Product details
- Search products
- Add to cart
- Remove from cart
- Increase/decrease quantity
- Checkout system
- Lazy loading
- Responsive design


## API

Products fetched from:

https://dummyjson.com/products


## Installation

Clone repository

npm install

npm run dev
